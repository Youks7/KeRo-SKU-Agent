# Contributing

Thanks for helping improve KeRo SKU Skill.

This repository is focused on real-SKU e-commerce page planning, product-fidelity rules, and safer AI image prompt workflows.

## What You Can Contribute

- Fix unclear documentation.
- Add product-page workflow examples.
- Improve installation or troubleshooting notes.
- Suggest safer product-fidelity rules.
- Add category-specific usage notes.
- Report cases where the Skill changes product identity or invents product facts.

## What Not To Submit

- Competitor images, product images, logos, or brand assets you do not have permission to share.
- Prompts that copy another brand's product, text, layout, or packaging.
- Claims that invent material, certification, medical effect, safety rating, user reviews, sales volume, or platform ranking.
- Large unrelated rewrites that change the Skill's core purpose.

## Example Contribution Format

When adding an example, include:

- Product category.
- Target platform.
- Input material required.
- Starter prompt.
- Stage 1 expected output.
- Stage 2 direction-selection example.
- Stage 3 per-screen production example.
- Product-fidelity QA checklist.

## Pull Request Checklist

- The change is related to SKU detail-page planning or product-image workflows.
- Links are relative and clickable.
- Markdown tables render correctly.
- Code fences are closed.
- No private product images or unauthorized competitor assets are included.
- Safety boundaries are preserved.

## Reporting Issues

Please include:

- What you tried to do.
- What product category was involved.
- Which stage failed.
- What the Skill output incorrectly.
- Whether the issue involved hallucinated facts, product deformation, unsafe competitor reference, or unclear documentation.
