# 平台路由表

| 用户上下文 | 使用 Skill | 典型素材槽位 |
| --- | --- | --- |
| 淘宝、淘系普通店铺 | `$sku-taobao` | 搜索主图、轮播图、SKU 属性图、详情模块 |
| 天猫、旗舰店、品牌商品 | `$sku-tmall` | 品牌主图、SKU 图、品牌详情、资质模块 |
| 拼多多、多多、拼购 | `$sku-pinduoduo` | 曝光图、轮播图、SKU 图、详情图 |
| 京东、JD | `$sku-jd` | 主图、附图、参数、兼容性、服务详情 |
| 1688、批发、工厂、OEM | `$sku-1688` | 采购页、阶梯价、定制、交付、询盘 |
| Amazon、ASIN、A+ | `$sku-amazon` | Main image、secondary images、A+、Brand Story |
| Shopify、独立站、PDP | `$sku-shopify` | 响应式 section、媒体、变体、SEO、CTA |
| TikTok Shop | `$sku-tiktok-shop` | PDP 图、变体图、Shop Ads 源图、内容交接 |

“抖音商城”不等于 TikTok Shop；“独立站”默认 Shopify 只是推荐而不是事实；Amazon 必须继续识别具体站点。

多平台任务不要先生成通用版本再翻译。公共事实相同，素材结构分别生成。
