下载 Skill 完整规则 Examples GitHub
01

产品分析

识别真实产品、事实边界、风险和视觉机会。

02

方向提案

给出 A / B / C 三种明显不同的详情页方向。

03

逐屏生产

确认方向后输出逐屏方案、Prompt 和质检点。

适合场景

  • 淘宝、天猫、京东、拼多多详情页
  • 抖音商城、小红书商品图文页
  • Amazon A+、Shopify、TikTok Shop 商品图

核心原则

  • 真实产品图是唯一产品身份来源
  • 不编造材质、规格、认证、功效和评价
  • 竞品图只参考结构，不复制品牌资产
