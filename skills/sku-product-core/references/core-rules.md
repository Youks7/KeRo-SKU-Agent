# 产品事实与保真规则

## 1. 识别输入

为每张输入图标记用途、视角、清晰度、是否包含包装、Logo、标签、配件和变体。区分用户真实 SKU 图、竞品参考图、风格参考图和 AI 概念图。

## 2. 建立事实层级

- 已确认事实：图片清晰可见、用户明确提供或文件证明。
- 谨慎推断：用于策略判断但不能写成正式卖点。
- 禁止主张：缺少证据或属于高风险承诺。
- 缺失证据：会影响平台发布、素材真实性或用户选择的信息。

不要从视觉外观直接断言具体材质、成分、功率、容量、兼容性、功效、认证或产地。

## 3. 选择处理模式

- 模式 A：品牌、高客单、主图、属性图、包装文字敏感或结构必须完全一致。
- 模式 B：真实参考图基础上的背景替换、光影优化或有限场景化；必须验证一致性。
- 模式 C：无实物的方向探索；明确不可作为真实 SKU 成品图。

普通 SKU 不代表可以自动使用模式 B。素材槽位或平台禁止数字渲染时使用模式 A。

## 4. 提炼视觉记忆点

每个记忆点都要包含：具体细节、证据位置、为什么属于本产品、适合的素材槽位。禁止把“高级、干净、科技、品质”单独作为记忆点。

## 5. 输出原则

先给关键结论，再给证据和风险。默认保持简洁，但不得省略事实边界、处理模式、禁止修改项和缺失证据。
