# 公共生产质检

## 事实检查

- 标题、图像、属性、参数、包装和文案是否描述同一个 SKU。
- 所有数字、材质、功效、认证和服务承诺是否有证据。
- 推断是否被误写成事实。
- 是否存在用户未提供的奖项、评价、排名或官方身份。

## 产品一致性

- 产品颜色、结构、比例和数量是否与真实图一致。
- Logo、包装文字、标签和配件是否被修改、添加或删除。
- 是否出现相似款替换、额外部件或错误变体。
- AI 编辑痕迹是否会让消费者误解实物。

## 画面质量

- 每个素材槽位是否完成独立商业任务。
- 构图是否保留平台要求的产品区和文字安全区。
- 连续画面是否改变镜头距离、角度、产品状态、背景或道具逻辑。
- Prompt 是否包含至少两个本产品专属细节。
- 把产品名替换为同类商品后仍完全成立的 Prompt 必须重写。

## 发布结论

把每个素材标记为以下状态之一：

- `ready`：事实、身份和平台规则均已核对。
- `human-review`：需要人工检查文字、合成、颜色或类目规则。
- `direction-only`：仅供方向参考，不能作为真实销售素材。
- `blocked`：缺少影响真实性或合规性的关键证据。
