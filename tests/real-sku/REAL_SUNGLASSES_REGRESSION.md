# 真实墨镜外部夹具回归

## 目的

用同一真实 SKU 的正面、四分之三侧面和背面商品图验证素材入口、多视图身份锚点、三方向策划、F0–F3 路由和人工审核闸门。私有商品图只在本机原位读取，不复制到公开仓库，也不把绝对路径写入测试记录。

## 外部输入

- 三张独立 PNG：正面、四分之三侧面、背面。
- 每张图短边不少于 800 px，文件大于 50 KB，SHA-256 不重复。
- 人工目视确认三张图属于同一白色大框墨镜变体：灰色渐变镜片、宽鼻梁、银色圆环装饰、白色镜腿、镜腿菱格纹理一致。
- 未从图片推断偏光、UV、材质、尺寸、认证、品牌或功效。

## 执行记录

2026-07-16 在仓库外的用户自有素材上执行：

```powershell
python scripts/validate_real_sku_fixture.py `
  --front "<private-front.png>" `
  --three-quarter "<private-three-quarter.png>" `
  --back "<private-back.png>"
```

检查结果：三视图均通过真实文件、分辨率、体积和独立哈希门槛；图片没有被复制进仓库。

## 行为回归

- 身份合同必须锁定镜片外轮廓、鼻梁、镜框色、渐变镜片色调、圆环装饰位置、镜腿外形和菱格纹理。
- 三个方向必须真正改变命题、视觉世界、商品角色、镜头光线和叙事中的至少三个维度，不能只换背景色。
- F0 用于变体和证据槽位；F1 用真实商品像素合成创意环境；F2 只有在多视图工具可用并通过人工身份审核时才能生产新机位或佩戴画面；F3 只用于方向探索。
- 任何新增 Logo、改变装饰形态、镜片渐变漂移、左右结构不一致、虚构人物佩戴尺度或错误反射都进入 `repair-required`。

## 诚实边界

该回归证明真实多视图素材能够进入 Agent 的身份合同、创意策划和生产路由，并不证明任意图像模型一定能生成同款成图。F2 的最终身份保真仍必须对实际生成结果进行人工对照审核；没有生成工具或没有实际成图时，不得把 F2 标为 `ready`。
